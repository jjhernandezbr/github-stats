## Description
CLI application that can be used to obtain statistics related to github 
usage and activity of each user inside an organization. 

## Installation

```bash
$ npm install
$ docker-compose up -d
$ npm run typeorm migration:run -- -d src/data-source
```

## Running the app

```bash
# Execute applicataion
$ npm run start
```

## Test

```bash
# unit tests
$ npm run test
```
