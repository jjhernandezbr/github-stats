## Installation

```bash
$ npm install
$ docker-compose up -d
$ npm run typeorm migration:run -- -d src/data-source
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
```
